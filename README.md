# Crowd Surge

## A minimalistic crowd funding web app

<br>

![image](./images/home.png)

## Features

- Create new campaigns
- Support existing campaigns by funding ether (ETH)
- Minimalistic UI
- Mobile responsive

## Tech Stack

- Frontend
    - Language: Typescript
    - Framework: React
    - Bundler: vite
    - CSS Framework: Tailwind

- Backend
    - Blockchain: Ethereum (goerli and sepolia)
    - Framework: Hardhat
